# BEEEER - Auzem'Beers & Sons

Welcome to the official repository for the **Auzem'Beers & Sons** website. This project showcases a local artisanal brewery based in Taller, Landes, France.

## 🍺 About the Project

This website is designed to provide an immersive experience for beer enthusiasts, featuring our unique brews and professional services.

### Key Features:
- **Age Verification:** Ensuring responsible access to our content.
- **Dynamic Product Showcase:** Explore our range of artisanal beers with detailed information.
- **Service Management:** Rent draft systems or book tasting sessions.
- **Responsive Design:** Optimized for all devices with a modern dark/light mode toggle.
- **User Accounts:** Manage your orders and preferences easily.

## 🚀 Technologies Used

- **HTML5:** Semantic structure for better accessibility and SEO.
- **CSS3:** Custom styles with modern features like CSS variables and flexbox/grid.
- **JavaScript:** Interactive components including theme switching and scroll animations.
- **Font Awesome:** High-quality icons for a polished look.

## 📂 Project Structure

- `index.html`: The homepage with our story and featured services.
- `produits.html`: Our complete beer catalogue.
- `commandes.html`: Interface for placing and managing orders.
- `service.html`: Information about rental and tasting events.
- `FAQ.html`: Frequently asked questions.
- `Avis.html`: Customer testimonials.
- `compte.html`: User profile and settings.
- `styles.css`: Core styling and theming logic.

## 🛠️ Development

This project was improved to ensure a professional and cohesive user experience, following modern web development standards.
